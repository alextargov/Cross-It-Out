    How to use

    Object: The object of the game is collect points as possible as you can.

    Equipment: Each account consist of starting 50 bonus points.

    Setup: Choose name, time, date and priority of the task, then click add task into some of categories.

    Playing the Game: Once the task is choosen your purpose is to finished it before end of the day. If your task is completed before end time you will win points based on priority(ps: you will be able to receive more points if you finish high priority tasks). Else these points will be substracted from accounts points.

    Please note: Negative points are possible.
